## How to sign

By submitting a pull request to this repository, you indicate that you have read
and agree to this CLA. For an explicit record, add the following line to your
commit message (using `git commit -s` adds the `Signed-off-by` automatically):

```
Signed-off-by: Your Name <your.email@example.com>
```

If contributing on behalf of an employer or other entity, an authorized
representative should state so in the pull request.
